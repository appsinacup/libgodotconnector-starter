# StarterApp for LibGodot Connector

Runs LibGodot inside Elixir.

## Installation

Get deps:

```sh
mix deps.get
```

Compile it:

```sh
mix deps.get
mix compile
```

Run it:

```sh
mix run --no-halt
```

We use no-halt because otherwise the app will exit.

Optional demo request:

```sh
iex -S mix
```

Then run `StarterApp.hello()`.
